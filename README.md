# Formulir Pendaftaran Komunitas Teknologi

Proyek ini berisi latihan pembuatan halaman web menggunakan **HTML dan CSS dasar**.
Halaman ini mencakup:
- Formulir pendaftaran dengan berbagai jenis input
- Penggunaan elemen `<span>` untuk memberikan gaya khusus pada teks
- Penggunaan elemen `<table>` untuk menampilkan data anggota

### ğŸ”¸ Elemen `<span>`
Digunakan untuk memberi gaya khusus pada bagian teks tertentu tanpa membuat baris baru.  
Contoh dari halaman:

### ğŸ”¹ Elemen `<table>`
Digunakan untuk menampilkan data dalam bentuk tabel.  
Contoh dari halaman:
```html

    <th>Nama</th>

  </tr>

    <td>AI</td>
```
